# Audit — ProRL: Prolonged Reinforcement Learning Expands Reasoning Boundaries in Large Language Models (NeurIPS 2025, #543)

## No code present — excluded from the reproducibility audit

This paper releases **no author code for the work**, so there is nothing to
reproduce or audit against. The only repository in the audit bundle, `open-thought/reasoning-gym`, is a third-party dependency (reference [19] — a community RL task suite by a different org), **not** the authors' ProRL implementation; the paper's sole released artefact is model weights (HuggingFace `Nemotron-Research-Reasoning-Qwen-1.5B`).

Because the study population is restricted to papers that release **the authors'
own code for the paper**, this paper contributes **no findings**. The prior audit
outputs — which only documented the absence of code — have been retired to
[`_superseded_audit/`](_superseded_audit/) for the record and are not counted in
any figure or aggregate.

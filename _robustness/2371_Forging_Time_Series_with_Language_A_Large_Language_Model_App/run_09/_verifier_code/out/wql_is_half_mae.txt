TTM emits a single point forecast, and compute_wql passes that same point
prediction for q=0.1, 0.5, 0.9. The pinball loss with one prediction is
0.5*|e| averaged over the symmetric quantile set, so the Table-2 'WQL'
column is exactly 0.5*MAE, not a Weighted Quantile Loss (which requires
distinct quantile/probabilistic forecasts).
